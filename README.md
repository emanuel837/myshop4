# myShop 4

Internal employee portal (shoe store chain), built as a PWA in Hebrew (RTL).

## Tech

- React + Vite
- Tailwind CSS
- PWA via `vite-plugin-pwa`

## Getting started

Install dependencies, then run dev server:

```bash
npm install
npm run dev
```

## Notes

- `index.html` sets `lang="he"` and `dir="rtl"`.
- This repository currently contains only the base project shell (no screens yet).
